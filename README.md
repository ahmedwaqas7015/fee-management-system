# Fee Management System (FMS)

A comprehensive Fee Management System for educational institutions built with Flask.

## Features

- Student Management
- Fee Management (Individual & Family Payments)
- Defaulters Management
- Receipt Generation
- Multi-language Support (Urdu & English)
- Excel Export
- And much more...

## Technology Stack

- **Backend**: Flask 2.3+
- **Database**: SQLite
- **ORM**: SQLAlchemy
- **Frontend**: Bootstrap 5, Jinja2
- **i18n**: Flask-Babel

## Installation

See installation guide in documentation.

## Development

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On Linux/Mac:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the application
python run.py
```
